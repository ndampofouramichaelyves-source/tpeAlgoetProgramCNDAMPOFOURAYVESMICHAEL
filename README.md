# Projet TPE  de NDAM POFOURA YVES MICHAEL - Bibliothèque Mathématique en C

## Description
Ce projet implémente une bibliothèque mathématique en algoritmhique et langage C avec diverses fonctions.

## Fonctionnalités implémentées

### Fonctions de base
1. `fact(n)` - Calcul de la factorielle
2. `power(x, n)` - Calcul de la puissance

### Exercices TPE
3. `permut(a, b)` - Permutation de deux entiers
4. `resoudreEquationSecondDegre(a, b, c)` - Résolution d'équation du second degré
5. `estPremier(n)` - Vérification si un nombre est premier
6. `distancePoints(p1, p2)` - Calcul de distance entre deux points
